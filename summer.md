# Javascript 編

実行に関していくつか方法はあるが今回は`.html`ファイルと`.js`ファイルを別個に用意するやり方について


- `.html`の名前を`practice.html`, `.js`ファイルの名前を`pr.js`として今回は扱う.
- `practice.html`として以下の内容を保存する

``` html:practice.html
<!DOCTYPE html>
<html lang = 'ja'>
<head>
  <title> タイトル名 </title>
  <meta charset="UTF-8">
</head>
<body>
  <script src = pr.js> </script>
</body>
</html>
```
